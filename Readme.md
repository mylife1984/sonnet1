This project is just for learning expressjs web framework.

# Install dependencies
	
	$ npm install express ejs
	
# Checkout the source

	$ git clone git://github.com/fzx/sonnet1.git
	 
# Running locally

	$ cd sonnet1
	
	$ node server.js
    
   You can see the home page from http://localhost:3000. 
	 
# Running all tests

	$ cd sonnet1
	
	$ ./test-all
